# 🌐 XRP Ledger Live Visualization

En avanceret, real-time 3D-visualisering af XRP Ledger (XRPL) netværket, bygget med Three.js. Se transaktioner flyve over globussen i real-time, direkte fra kilden.

### 🚀 [Se Live Demo Her!](https://goprogabriel.github.XRP/)
*(Husk at aktivere GitHub Pages i dine indstillinger for at se den på dit eget domæne)*

---

## ✨ Funktioner
- **Live 3D Globe**: Se transaktioner i real-time som lysende buer over hele verden.
- **⚡ Netværksstatistik**: Live TPS (Transactions Per Second) og Ledger Close Time (LCT) direkte i dashboardet.
- **🔍 Adresse Search**: Indtast en XRP-adresse for at fremhæve specifikke transaktioner med hvidt lys.
- **🐋 Whale Filter**: Filtrer støjen fra og se kun de helt store bevægelser (>10.000 XRP).
- **🔧 Validator Network**: Se de fysiske lokationer af de vigtigste XRP validator nodes.
- **🎥 Interaktive Kontroller**: Auto-rotation, manuel zoom og nulstil fokus.

## 🛠️ Teknologier
- **Three.js**: Til den høj-performante 3D-modellering.
- **WebSockets**: Direkte forbindelse til `xrplcluster.com` for minimal latency.
- **Vanilla CSS/JS**: Ingen tunge frameworks, lynhurtig indlæsning.

## 📦 Installation & Hosting
Da dette er en rent frontend-baseret applikation, kræver den ingen backend!

1. **Lokalt**: Åbn blot `index.html` i din browser.
2. **GitHub Pages**:
   - Gå til dit repo på GitHub.
   - Gå til **Settings > Pages**.
   - Vælg **Branch: main** og tryk **Save**.
